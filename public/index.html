Path HTTP Methods Description

HTTP Methods for Users

Path HTTP Method Description
/ GET Retrieves list of users
/:id GET Retrieves a specific user
/register POST Creates a new user
/updateUser/:id PATCH Updates details of a specific user
/deleteUser/:id DELETE Deletes a specific user
/login POST Logs user in
